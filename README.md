# NetFlix-Clone

This project is a Netflix Clone developed using HTML, CSS, and JavaScript to recreate the look and feel of the Netflix homepage. It focuses on responsive design, clean UI structure, modern styling, and basic interactivity to improve front-end development skills.

A fully responsive Netflix Clone built with HTML, CSS, and JavaScript, designed to replicate the visual experience of Netflix with modern UI components, interactive sections, and mobile-friendly responsiveness.
